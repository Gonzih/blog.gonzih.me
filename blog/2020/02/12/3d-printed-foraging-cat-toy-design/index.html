3D printed foraging cat toy design

Wed, Feb 12, 2020

Here is a foraging cat toy based on design I saw in youtube video by EvanAndKatelyn.

Main difference in this design is that it’s implemented in OpenSCAD and as a result fully configurable.

3D designs are available on thingiverse or github. OpenSCAD source code is available also on github. One can use either OpenSCAD or thingiverse to customize the design.
