3D printed foraging cat toy

Wed, Feb 12, 2020

Here is a foraging cat toy based on design I saw in a youtube video by EvanAndKatelyn.

Main difference in this design is that it’s implemented in OpenSCAD and as a result is fully configurable.

3D designs are available on github. OpenSCAD source code is available also on github, which can be used to customize this design.
